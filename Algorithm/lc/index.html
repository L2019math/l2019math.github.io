  1. 枚举子串开头位置 iii，然后枚举子串结束位置 jjj，动态计算出当前子数组组成的字符串 sss。
  2. 注意，组成 sss 的过程需要在每一个数字后添加一个特殊分隔符来区分，避免冲突。

第四题

方法一: DP

  1. 提示 1-1
    将所有子串按照其末尾字符的下标分组。

  2. 提示 1-2
    考虑两组相邻的子串：以 s[i-1]s[i−1] 结尾的子串、以 s[i]s[i] 结尾的子串。

  3. 提示 1-3
    以 s[i]s[i] 结尾的子串，可以看成是以 s[i-1]s[i−1] 结尾的子串，在末尾添加上 s[i]s[i] 组成。

上面这一串提示是思考子串统计类问题的通用技巧之一。

从左往右遍历 sss，考虑将 s[i]s[i]s[i] 添加到以$ s[i-1]$ 结尾的子串的末尾。添加后，这些子串的引力值会增加多少？

  • 如果 s[i]s[i]s[i] 之前没有遇到过，那么这些子串的引力值都会增加 1，这些子串的引力值之和会增加 iii，再加上 1，即 s[i]s[i]s[i]单独组成的子串的引力值；
  • 如果$ s[i]$ 之前遇到过，设其上次出现的下标为 ，那么向子串s[0..i−1], s[1..i−1], s[2..i−1],⋯ ,s[j..i−1]s[0..i-1],\ s[1..i-1],\ s[2..i-1],\cdots,s[j..i-1]s[0..i−1], s[1..i−1], s[2..i−1],⋯,s[j..i−1] 的末尾添加 s[i]s[i] 后，这些子串的引力值是不会变化的，因为 s[i]s[i]s[i]已经在 s[j]s[j]s[j]处出现过了；而子串 s[j+1..i−1], s[j+2..i−1],⋯ ,s[i−1..i−1]s[j+1..i-1],\ s[j+2..i-1],\cdots,s[i-1..i-1]s[j+1..i−1], s[j+2..i−1],⋯,s[i−1..i−1] 由于不包含字符 s[i]s[i]s[i]，这些子串的引力值都会增加 1，因此有 i−j−1i-j-1i−j−1 个子串的引力值会增加 1，这些子串的引力值之和会增加 i−j−1i-j-1i−j−1，再加上 1，即 s[i]s[i]s[i] 单独组成的子串的引力值。

模拟上述过程，遍历 sss 的过程中用一个变量$ \textit{sumG} $维护以 s[i]s[i]s[i] 结尾的子串的引力值之和，同时用一个数组 last\textit{last}last 记录每个字符上次出现的下标。

累加遍历中的 sumG\textit{sumG}sumG 即为答案。

方法二： (思维题) O(n+∣Σ∣)O(n + |\Sigma|)O(n+∣Σ∣)

  1. 独立统计每个字符的贡献。
  2. 进一步逆向思维，统计子串中未出现当前字符的数量，然后用全量子串减去这个数量就是这个字符的贡献。
  3. 未出现当前字符的子串一定出现在相邻两个该字符的位置之间，以及首次出现的位置之前以及最后一次出现的位置之后。

时间复杂度

  • 遍历字符串一次，遍历所有字符一次，故总时间复杂度为 O(n+∣Σ∣)O(n + |\Sigma|)O(n+∣Σ∣)。

空间复杂度

  • 需要 O(∣Σ∣)O(|\Sigma|)O(∣Σ∣) 的额外空间存储每个字符的出现位置。
