problem 218

求极限 lim⁡x→1 x−xx1−x+ln⁡x\text{求极限\ }\lim _{x\rightarrow 1}\ \frac{x-x^x}{1-x+\ln x} 求极限 x→1lim​ 1−x+lnxx−xx​

  • 思路：

    幂指函数形式取对数，趋向于1，利用+1-1转换为趋向于零

  • 具体过程：

    lim⁡x→1  x−xx1−x+ln⁡x=  lim⁡x→1exlnx(ln⁡x−xln⁡x)1−x+ln⁡x=lim⁡x→1ln⁡x(1−x)1−x+ln⁡x=lim⁡u→0  −uln⁡(1+u)−u+ln⁡(1+u)=lim⁡u→0  −u2−12u2=2\begin{align*} &\lim_{x\rightarrow 1}\,\,\frac{x-x^x}{1-x+\ln x} \\ &=\,\,\lim_{x\rightarrow 1}\frac{e^{xlnx}\left( \ln x-x\ln x \right)}{1-x+\ln x}\\ &=\lim_{x\rightarrow 1}\frac{\ln x\left( 1-x \right)}{1-x+\ln x}\\ &=\lim_{u\rightarrow 0}\,\,\frac{-u\ln \left( 1+u \right)}{-u+\ln \left( 1+u \right)}=\lim_{u\rightarrow 0}\,\,\frac{-u^2}{-\frac{1}{2}u^2} \\ &=2 \end{align*} ​x→1lim​1−x+lnxx−xx​=x→1lim​1−x+lnxexlnx(lnx−xlnx)​=x→1lim​1−x+lnxlnx(1−x)​=u→0lim​−u+ln(1+u)−uln(1+u)​=u→0lim​−21​u2−u2​=2​

经典错误：不能对复函函数等价替换，要对因子等价替换

lim⁡x→0(1+x)2x−e2[1−ln⁡(1+x)]x=  e2lim⁡x→0eln⁡(1+x)−xx−1+ln⁡(1+x)x=  e2lim⁡x→02ln⁡(1+x)−xx+x+O(x)x= e2 ∗ 0 =0\begin{align*} &\lim_{x\rightarrow 0}\frac{\left( 1+x \right) ^{\frac{2}{x}}-e^2\left[ 1-\ln \left( 1+x \right) \right]}{x} \\&=\,\,e^2\lim_{x\rightarrow 0}\frac{e^{\frac{\ln \left( 1+x \right) -x}{x}}-1+\ln \left( 1+x \right)}{x} \\&=\,\,e^2\lim_{x\rightarrow 0}\frac{2\frac{\ln \left( 1+x \right) -x}{x}+x+O\left( x \right)}{x} \\&=\ e^2\ *\ 0\ =0 \end{align*} ​x→0lim​x(1+x)x2​−e2[1−ln(1+x)]​=e2x→0lim​xexln(1+x)−x​−1+ln(1+x)​=e2x→0lim​x2xln(1+x)−x​+x+O(x)​= e2 ∗ 0 =0​

problem 220

  • 思路

    幂指型函数取对数，“左提右式”+等价无穷小代换

  • 具体过程

lim⁡x→0  (1+x)1x−(1+2x)12xsin⁡x=  lim⁡x→0  eln⁡(1+2x)2x  2ln⁡(1+x)−ln⁡(1+2x)2xx=  lim⁡x→0  12x+O(x)x=12\begin{align*} &\lim_{x\rightarrow 0}\,\,\frac{\left( 1+x \right) ^{\frac{1}{x}}-\left( 1+2x \right) ^{\frac{1}{2x}}}{\sin x} \\&=\,\,\lim_{x\rightarrow 0}\,\,\frac{e^{\frac{\ln \left( 1+2x \right)}{2x}}\,\,\frac{2\ln \left( 1+x \right) -\ln \left( 1+2x \right)}{2x}}{x} \\&=\,\,\lim_{x\rightarrow 0}\,\,\frac{\frac{1}{2}x+O\left( x \right)}{x}=\frac{1}{2} \end{align*} ​x→0lim​sinx(1+x)x1​−(1+2x)2x1​​=x→0lim​xe2xln(1+2x)​2x2ln(1+x)−ln(1+2x)​​=x→0lim​x21​x+O(x)​=21​​

problem 222

求极限 lim⁡x→0 xsin⁡x2−2(1−cos⁡x)sin⁡xx5\text{求极限\ }\lim _{x\rightarrow 0}\ \frac{x\sin x^2-2\left( 1-\cos x \right) \sin x}{x^5} 求极限 x→0lim​ x5xsinx2−2(1−cosx)sinx​

  • 思路

    分母是五阶，对分子Taylor展开到五阶

  • 具体过程

lim⁡x→0  xsin⁡x2−2(1−cos⁡x)sin⁡xx5=  lim⁡x→0  x(x2−16x6)−2(12x2−124x4+16!x6)(x−16x3+15!x5)+O(x5)x5=  lim⁡x→0  14x5+O(x5)x5=14\begin{align*} &\lim_{x\rightarrow 0}\,\,\frac{x\sin x^2-2\left( 1-\cos x \right) \sin x}{x^5} \\&=\,\,\lim_{x\rightarrow 0}\,\,\frac{x\left( x^2-\frac{1}{6}x^6 \right) -2\left( \frac{1}{2}x^2-\frac{1}{24}x^4+\frac{1}{6!}x^6 \right) \left( x-\frac{1}{6}x^3+\frac{1}{5!}x^5 \right) +O\left( x^5 \right)}{x^5} \\&=\,\,\lim_{x\rightarrow 0}\,\,\frac{\frac{1}{4}x^5+O\left( x^5 \right)}{x^5}=\frac{1}{4} \end{align*} ​x→0lim​x5xsinx2−2(1−cosx)sinx​=x→0lim​x5x(x2−61​x6)−2(21​x2−241​x4+6!1​x6)(x−61​x3+5!1​x5)+O(x5)​=x→0lim​x541​x5+O(x5)​=41​​
