problem 223

设 $f’(0)=0,f’’(0)存在$，求极限 $\lim\limits_{x\rightarrow 0}\frac{f\left( x \right) -f\left( \ln \left( 1+x \right) \right)}{x^3}$

  • 思路

    带拉格朗日余项的麦克劳林公式展开 ×

    求导之后利用导数的定义 ×

    拉格朗日之后利用导数定义 + 夹逼 √

  • 具体过程

    考察导数定义的灵活运用，并不是只有 x 趋近于 0 时才能用，任何变量（包含拉格朗日产生的变量）趋近于 0 时，形式符合导数定义时都可以用。

    \&=\,\,\lim_{x\rightarrow 0^+}\frac{1}{x\sqrt{x}}\left( \sqrt{a}\left( \arctan \sqrt{\frac{x}{a}}-\sqrt{\frac{x}{a}}+\sqrt{\frac{x}{a}} \right) -\sqrt{b}\left( \arctan \sqrt{\frac{x}{b}}-\sqrt{\frac{x}{b}}+\sqrt{\frac{x}{b}} \right) \right)

    \&=\,\,\lim_{x\rightarrow 0^+}\frac{1}{x\sqrt{x}}\left( \sqrt{a}\left( -\frac{1}{3}\left( \sqrt{\frac{x}{a}} \right) ^3+\sqrt{\frac{x}{a}} \right) -\sqrt{b}\left( -\frac{1}{3}\left( \sqrt{\frac{x}{b}} \right) ^3+\sqrt{\frac{x}{b}} \right) \right)

    \&=\,\,\lim_{x\rightarrow 0^+}\frac{1}{x\sqrt{x}}\left( -\frac{x\sqrt{x}}{3a}+\frac{x\sqrt{x}}{3b} \right)

\&=\ \frac{a-b}{3ab}

\end{align*}
