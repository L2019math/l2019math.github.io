题目151

设 f(x,y,z)=ex+y2z+sin⁡(y−1)x2+z2f( x,y,z ) =e^x+y^2z+\frac{\sin ( y-1 )}{\sqrt{x^2+z^2}}f(x,y,z)=ex+y2z+x2+z2​sin(y−1)​ ，且对任意 $ z=z(x,y) $ 是由方程 $ x+y+z+2xyz=0$ 所确定的隐函数，求 $f_x’\left( 0,1,-1 \right) $

解答

考察偏导数的计算

原函数过于复杂且求具体点的偏导值，考虑先代后求

f(x,1,z)=ex+z(1)f(x,1,z)=e^x+z \tag1 f(x,1,z)=ex+z(1)

对（1）式对 x 求偏导,得

fx′(x,1,z)=ex+∂z∂x(2)f_x'(x,1,z)=e^x+\frac{\partial z}{\partial x} \tag2 fx′​(x,1,z)=ex+∂x∂z​(2)

隐函数对x求偏导，得

1+∂z∂x+2y(z+x∂z∂x)=0(3)1+\frac{\partial z}{\partial x}+2y\left( z+x\frac{\partial z}{\partial x} \right) =0 \tag3 1+∂x∂z​+2y(z+x∂x∂z​)=0(3)

将x=0,y=1,z=-1 代入（3）式，得 ∂z∂x=1\frac{\partial z}{\partial x}=1∂x∂z​=1

代入（1）式，得 fx′(0,1,−1)=2f_x'\left( 0,1,-1 \right) =2fx′​(0,1,−1)=2

齐次函数

这里给出齐次函数的概念

[定理] 若对任意 t>0t>0t>0 有 f(tx,ty)=tnf(x,y)f(tx,ty)=t^nf(x,y)f(tx,ty)=tnf(x,y)，则称函数 $ f(x,y)$ 是 $ n$ 次齐次函数，若 f(x,y)f(x,y)f(x,y) 可微，则

f(x,y)是n次齐次函数⟺x∂f∂x+y∂f∂y=nf(x,y)f(x,y) 是 n 次齐次函数 \Longleftrightarrow x\frac{\partial f}{\partial x}+y\frac{\partial f}{\partial y}=nf(x,y) f(x,y)是n次齐次函数⟺x∂x∂f​+y∂y∂f​=nf(x,y)

题目153

∂2z∂x∂y=exy+xyexy+f11′′+(x+y)f12′′+f2′+xyf22′′\frac{\partial ^2z}{\partial x\partial y}=e^{xy}+xye^{xy}+f_{11}''+(x+y)f_{12}''+f_2'+xyf_{22}'' ∂x∂y∂2z​=exy+xyexy+f11′′​+(x+y)f12′′​+f2′​+xyf22′′​
