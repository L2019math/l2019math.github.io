解答

题目165

∫01dy∫y1x2−y2dx\int_{0}^{1}dy\int_{y}^{1}\sqrt{x^2-y^2} dx ∫01​dy∫y1​x2−y2​dx

解答

考察交换积分次序

交换积分次序得，∫01dx∫0xx2−y2dy\int_{0}^{1}dx\int_{0}^{x}\sqrt{x^2-y^2} dy∫01​dx∫0x​x2−y2​dy

对内层积分来说，x 是常数，考虑利用几何意义 ∫0aa2−y2dy=14πa2\int_{0}^{a}\sqrt{a^2-y^2} dy = {\frac{1}{4}}\pi a^2∫0a​a2−y2​dy=41​πa2

∫01dx∫0xx2−y2dy=∫0114πx2dx=112π\int_{0}^{1}dx\int_{0}^{x}\sqrt{x^2-y^2} dy=\int_{0}^{1} {\frac{1}{4}}\pi x^2dx={\frac{1}{12}}\pi ∫01​dx∫0x​x2−y2​dy=∫01​41​πx2dx=121​π

题目166

∫01dy∫y1(ex2x−ey2)dx\int_{0}^{1}dy\int_{y}^{1}{(\frac{e^{x^2}}{x}-e^{y^2})} dx ∫01​dy∫y1​(xex2​−ey2)dx

解答

考察交换积分次序

交换积分次序后得

∫01dx∫0xex2xdy−∫01dy∫y1ey2dx\int_{0}^{1}dx\int_{0}^{x}\frac{e^{x^2}}{x} dy-\int_{0}^{1}dy\int_{y}^{1}{e^{y^2}} dx ∫01​dx∫0x​xex2​dy−∫01​dy∫y1​ey2dx

进一步计算，发现被积函数含有 yey2,ex2ye^{y^2},e^{x^2}yey2,ex2 ,属于超越函数，无法直接求解，故挖掘题目信息，消掉该项

∫01ex2dx−∫01(1−y)ey2dy=∫01yey2dy=12(e−1)\int_{0}^{1}e^{x^2}dx-\int_{0}^{1}(1-y){e^{y^2}}dy =\int_{0}^{1}y{e^{y^2}}dy=\frac{1}{2}(e-1) ∫01​ex2dx−∫01​(1−y)ey2dy=∫01​yey2dy=21​(e−1)

题目167

设区域  D  由曲线  y=sinx,x=π2,x=−π2,y=1  围成，则∬D(xy5−1)dxdy设区域\,\,D\,\,由曲线 \,\,y=sinx,x=\frac{\pi}{2},x=-\frac{\pi}{2},y=1\,\, 围成，则\iint_D(xy^5-1)dxdy 设区域D由曲线y=sinx,x=2π​,x=−2π​,y=1围成，则∬D​(xy5−1)dxdy

  • （A） π\piπ
  • （B）2
  • （C）2
  • （D）π\piπ

解答一

直接积分+定积分的几何意义

∬Dxy5dxdy=∫−π2π2dx∫sinx1xy5dy=∫−π2π216x(1−sin6x)dx(1)\iint_Dxy^5dxdy=\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}}dx\int_{sinx}^1xy^5dy=\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}}\frac{1}{6}x(1-sin^6x)dx \tag1 ∬D​xy5dxdy=∫−2π​2π​​dx∫sinx1​xy5dy=∫−2π​2π​​61​x(1−sin6x)dx(1)

（1）式，注意到对称区间，且关于x 为奇函数，故（1）式为零

∬x2+y2 ≤ 1(4x2+9y2)dσ=∬x2+y2 ≤ 1(9x2+4y2)dσ=132∬x2+y2 ≤ 1(x2+y2)dσ\iint_{x^2+y^2\ \le \ 1}{\left( 4x^2+9y^2 \right) d\sigma}=\iint_{x^2+y^2\ \le \ 1}{\left( 9x^2+4y^2 \right) d\sigma}=\frac{13}{2}\iint_{x^2+y^2\ \le \ 1}{\left( x^2+y^2 \right) d\sigma} ∬x2+y2 ≤ 1​(4x2+9y2)dσ=∬x2+y2 ≤ 1​(9x2+4y2)dσ=213​∬x2+y2 ≤ 1​(x2+y2)dσ

再由几何意义

∬D(−1)dxdy=−S=−π\iint_D(-1)dxdy=-S=-\pi ∬D​(−1)dxdy=−S=−π

故结果为 −π-\pi−π

解法二

构造对称区间

由图所示，划分为红色区域和棕色区域

xy5xy^5xy5 在红色区域上关于x轴对称，且关于y为奇函数，故在红色区域上的积分为零

xy5xy^5xy5 在棕色区域上关于y轴对称，且关于x为奇函数，故在棕色区域上的积分为零

故而题目转化为求积分区域的面积的相反数

题目168

计算二重积分  ∬Dx2y  dxdy,其中  D  是由双曲线  x2−y2=1  及直线  y=0,y=1  所围成的平面区域计算二重积分 \,\, \iint_Dx^2y \,\,dxdy,其中 \,\, D \,\, 是由双曲线\,\, x^2-y^2=1 \,\, 及直线 \,\, y=0,y=1\,\, 所围成的平面区域 计算二重积分∬D​x2ydxdy,其中D是由双曲线x2−y2=1及直线y=0,y=1所围成的平面区域

解答

考察常见曲线的函数图像和二重积分的计算

积分区域关于y轴对称，积分函数关于x为偶函数

原式=2∫01dy∫0y2+1x2y  dx原式=2\int_0^1dy\int_0^{\sqrt{y^2+1}}x^2 y \,\,dx \\ 原式=2∫01​dy∫0y2+1​​x2ydx

计算可得，结果为 215(42−1)\frac{2}{15}(4\sqrt2-1)152​(42​−1)
