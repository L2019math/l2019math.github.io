常数K值法与中值可分离型问题

等式只有端点函数值，导数值有关的常数和中值点的函数值或导数值，考虑采用 常数 K 值法

适用条件：

  1. 区间端点与中值可 分离，即原式可化成左侧参数只含有端点，右侧参数只含有中值 $\xi$
  2. 把式中的b换作a时，原式呈 $ 0 = 0$ 形式（零式），或等式两端相等（对称式）

构造步骤：

  1. 分离中值和端点，令等式一端为常数K

  3. 该辅助函数必然满足，$ F(a)=F(b)$ ，利用罗尔定理求解即可

虚假的双中值问题

不要求 $\xi \ne \eta$

在同一区间上用两次中值定理（拉格朗日中值定理、柯西中值定理）

对于这类问题，我们可以化归到 中值可分离型 问题上来，分析中值$ \xi,\eta $于等式两侧，然后处理两侧：

  1. 对含有 $\xi$侧的式子：将含有$\xi$的式子化为$\dfrac{F’(\xi)}{G’(\xi)}$ 或 $F’(\xi)$然后还原成 $\dfrac{F(b)-F(a)}{G(b)-G(a)}$ 或 $\dfrac{F(b)-F(a)}{b-a}$

  2. 对含有 $\eta$侧的式子：将含有 $\eta$的式子化为$\dfrac{H’(\eta)}{T’(\eta)}$ 或 $H’(\xi)$然后还原成 $\dfrac{H(b)-H(a)}{T(b)-T(a)}$ 或 $\dfrac{H(b)-H(a)}{b-a}$

凑出两个等式成立的条件：成 $\dfrac{F(b)-F(a)}{G(b)-G(a)}=\dfrac{H(b)-H(a)}{T(b)-T(a)}$从而完成证明
